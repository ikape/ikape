# Contributing

Thank you for taking the time to contribute to this project!

## How to Contribute

### Getting Started
1. **Fork the repository** - Click the "Fork" button at the top right of this repository
2. **Clone your fork** - `git clone https://github.com/YOUR_USERNAME/ikape.git`
3. **Create a feature branch** - `git checkout -b feature/your-feature-name`

### Making Changes
- Make your changes in your feature branch
- Commit your changes with clear, descriptive commit messages
- Keep your branch up to date with the main branch: `git pull origin main`

### Submitting Changes
1. **Push your branch** - `git push origin feature/your-feature-name`
2. **Open a Pull Request** - Go to the original repository and click "New Pull Request"
3. **Describe your changes** - Provide a clear description of what you changed and why

## Guidelines
- Fork the repository
- Create a feature branch
- Open a pull request

## Code Style
- Follow existing code conventions
- Ensure code is readable and well-commented
- Write clear, descriptive commit messages
- Keep PRs focused on a single feature or fix
- Test your changes before submitting
- Be respectful and constructive in discussions
